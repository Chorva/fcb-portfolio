pwd - present working directory

ls - lists content of folder/directory

cd - change directory (example: cd Directory)

mkdir - make directory (example: mkdir fcb-portfolio)

touch - make file (example: touch index.html  touch style.css)

CLI Commands:

ls --> list content of folder

pwd --> display current location

cd <folder_name> --> go inside the target folder

mkdir <new_folder_name> --> create the new folder

touch <file_name> --> create the new file
